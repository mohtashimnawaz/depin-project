# Design Tokens (Minimal / Modern)

## Colors
- primary: #0f172a (charcoal)
- background: #ffffff (white)
- surface: #f1f5f9 (muted background)
- muted: #64748b (muted text)
- accent: #6366f1 (indigo)
- success: #10b981 (green)
- destructive: #ef4444 (red)
- border: #e6edf3

## Typography
- primary font: Inter, system-ui, -apple-system, 'Segoe UI', Roboto, 'Helvetica Neue', Arial
- headings: font-weight 700-800; H1 ~ 44px; H2 ~ 32px
- body: font-size 14-16px; line-height ~1.4

## Spacing
- base spacing: 4px
- scales: 4, 8, 12, 16, 20, 24, 32, 40

## Buttons
- primary: background primary, text white, radius 10px, padding 12px 20px
- secondary: background white, border 1px solid border, text primary

## Elevation
- card: subtle border + shadow: 0 1px 2px rgba(15, 23, 42, 0.04)

## Notes
- Prefer neutral layout with restrained use of accent color for CTAs and small UI elements.
- Ensure sufficient contrast for accessibility (WCAG AA minimum for body text).
